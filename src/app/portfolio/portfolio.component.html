Connect Four Game

Java

https://youtu.be/yB2WFNBMnzM

In-progress Project

Difficulty:

Connect four is a two player board game in which the players alternately drop colored disks into a seven column, six row vertically suspended grid. Ths objective of the game is to connect four same-coloured disks in a row, a column, or a diagonal before the opponent can do likewise.

The program prompts two players to drop a red or yellow disk alternately.

The most challeging part was...

Whenever a disk is droppped, the program redisplays the board on the console and determins the status of the game (win, draw, continue).

I have been working on my codes in order to handle "draw" and "continue" so far. I will develop some algorithms to deal with such diverse scenarios.

What I have learned is...

I have learned how to traverse diagonally in a 2 dimenstional array and how to get the arrays for the rows and the columns.
